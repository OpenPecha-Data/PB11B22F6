/ bdag daṅ chos su ñer 'dogs pa // sna tshogs dag ni gaṅ byuṅ ba // de ni rnam par śes par gyur
// gyur pa de yaṅ rnam gsum ste /

/ rnam par smin daṅ ṅar sems daṅ // yul la rnam par rig pa'o // de la kun gźi rnam śes ni //
rnam smin sa bon thams cad pa /

/ de ni len pa dag daṅ gnas // rnam par rig pa mi rig pa // rtag tu reg daṅ yid byed daṅ //
rig daṅ 'du śes sems par ldan /

/ de la tshor ba btaṅ sñoms te // de ni ma bsgribs luṅ ma bstan // reg la sogs pa'aṅ de bźin
no // de ni rgyun 'bab chu bo bźin /

/ dgra bcom ñid na de ldog go // de la gnas te rab 'byuṅ źiṅ // de la dmigs pa yid ces bya //
rnam śes ṅar sems bdag ñid can /

/ bsgribs la luṅ du ma bstan pa'i // ñon moṅs bźi daṅ rtag tu 'grogs // bdag tu lta daṅ bdag
tu rmoṅs // bdag rgyal bdag chags 'du śes pa /

/ gaṅ du skyes pa de'i'o gźan // reg sogs kyaṅ de dgra bcom med // 'gog pa'i sñoms par 'jug
la med / / 'jig rten 'das pa'i lam na'aṅ med /

/ 'di ni gyur pa gñis pa'o // gsum pa yul rnam drug po la // dmigs pa gaṅ yin de dag ste //
dge daṅ mi dge gñi ga min /

/ kun tu 'gro daṅ bye brag des // sems las byuṅ ba dge ba daṅ // de bźin ñon moṅs ñe ñon moṅs
// tshor ba gsum daṅ de mtshuṅs ldan /
