/ gźan gyi dbaṅ gi ṅo bo ñid // rnam rtog yin te rkyen las byuṅ // grub ni de la sṅa ma po //
rtag tu med par gyur pa gaṅ /

/ de phyir de ñid gźan dbaṅ las // gźan min gźan ma yin pa'aṅ min // mi rtag pa sogs bźin du
brjod // de ma mthoṅ bar de mi mthoṅ /

/ ṅo bo ñid ni rnam gsum gyis // ṅo bo ñid med rnam gsum la // dgoṅs nas chos rnams thams cad
kyi // ṅo bo ñid med bstan pa yin /

/ daṅ po pa ni mtshan ñid kyi // ṅo bo ñid med gźan pa yaṅ // de ni raṅ ñid mi 'byuṅ bas //
ṅo bo ñid med gźan yin no /

/ chos kyi don gyi dam pa'aṅ de // 'di ltar de bźin ñid kyaṅ de // dus rnams kun na'aṅ de bźin
ñid // de ñid rnam par rig pa tsam /

/ ji srid rnam rig tsam ñid la // rnam par śes pa mi gnas pa // 'dzin pa gñis kyi bag la ñal
// de ñid rnam par mi ldog go /

/ 'di dge rnam rig tsam ñid ces // de sñam du ni dmigs nas su // ci yaṅ ruṅ ste mdun 'jog na
// de ni tsam la mi gnas so /

/ nam źig śes pas dmigs pa rnams // mi dmigs de yi tshe na ni // rnam par rig pa tsam la gnas
// gzuṅ ba med pas de 'dzin med /

/ de ni sems med mi dmigs pa // 'jig rten 'das pa'i ye śes med // gnas kyaṅ gźan du gyur pa
ste // gnas ṅan len gñis spaṅs pa'o /

/ de ñid zag pa med daṅ dbyiṅs // bsam gyis mi khyab dge daṅ brtan // de ni bde ba rnam grol
sku // thub ba chen po'i chos źes bya /

/sum cu pa'i tshig le'ur byas pa slob dpon chen po dbyig gñen gyis mdzad pa rdzogs so //// rgya
